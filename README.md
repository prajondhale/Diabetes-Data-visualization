# Diabetes-Data-visualization


This project focuses on analyzing the Pima Indians Diabetes Dataset using Python.
The dataset was cleaned and explored to understand key health factors associated with diabetes.
Data preprocessing techniques such as handling duplicates and checking missing values were performed before analysis.
Various visualizations including histograms, bar charts, scatter plots, box plots, pair plots, and correlation heatmaps were created using Matplotlib and Seaborn. 
These visualizations help identify trends, relationships, and patterns within the dataset.



Technologi Used:
Python
Pandas
NumPy
Matplotlib
Seaborn


Key Features:
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Statistical Summary Generation
Data Visualization and Pattern Analysis
